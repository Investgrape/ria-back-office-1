# RIA Back Office

A modern compliance and operations dashboard for Registered Investment Advisers.

## Features

- Dashboard analytics
- Document management
- Email system
- Employee tracking
- Marketing tools
- Audit system
- Client agreements

## Tech Stack

- React/TypeScript
- Tailwind CSS
- React Router
- Lucide React

## Development

```bash
npm install
npm start
```

## Deployment

This project is deployed on Vercel with automatic deployments.
